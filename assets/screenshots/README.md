# Screenshots

This directory contains screenshots of the AI Essay Writer application for documentation purposes.

## Required Screenshots

To complete the documentation, please add screenshots of:

1. **login.png** - Authentication screen showing OAuth options
2. **writing.png** - Main essay writing interface with agents
3. **agents.png** - Agent workflow status and progress
4. **history.png** - Essay management and history view
5. **dashboard.png** - User dashboard overview
6. **mobile.png** - Mobile responsive design

## Image Guidelines

- **Format**: PNG or JPG
- **Resolution**: 1920x1080 or higher
- **Quality**: High quality, clear text
- **Browser**: Use Chrome or Firefox
- **Theme**: Light theme preferred
- **Content**: Use sample data, no real user information

## How to Take Screenshots

1. Run the application locally
2. Create sample essays and data
3. Navigate to each major section
4. Take full-page screenshots
5. Crop to show relevant UI elements
6. Save with descriptive filenames
